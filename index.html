This page is the start of a great website, but there's still some work to do. You can add your own pages by creating HTML files next to these ones, and adding the appropriate front matter (you can use the included pages as templates). To add your new pages to the menu at the top, open the _includes folder, then open the menubar.html file. Next, add a link to your page as a normal HTML link. If you would like a dropdown menu like we use here, copy the line for that and change the options to suit you.

Make sure you include your group's logo - the Scouts Brand Centre has an option to create a logo image. Export it as PNG, then save it to /assets/img with the name "logo.png". It will then overwrite the default logo, and your site will start to look your own. To carry on the branding for the rest of the theme, edit _config.yml to set your group name.

The Downloads and Stats pages are included and should be ready for use after you change the name in front matter to match your group's. The Downloads page pulls files from /downloads, so save any file you would like the users to be able to download to there.

The Documentation is available online, and can be accessed through the purple button below.
